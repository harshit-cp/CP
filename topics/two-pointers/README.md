## Overview

## Key Concepts
- Left-right pointers (converging)
- Slow-fast pointers

## Common Patterns
1. **Opposite Direction** - Start from both ends, move toward center
2. **Same Direction** - Both start from beginning, different speeds
3. **Sorted Array** - Leverage sorted property for efficient search

## When to Use
- Sorted arrays
- Finding pairs with specific sum/property
- Removing duplicates in-place
- Palindrome checking
- Container with most water type problems
